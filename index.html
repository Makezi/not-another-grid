Basics

Building a responsive grid layout is simple:

  1. Add the .row class as a container, surrounding your columns
  2. Add as many .col elements as you want

Each column will have equal width, no matter how many columns you insert into the row.

Sizes

You can individually change the size of a single column with the following classes:

  • .col-half
  • .col-one-third
  • .col-two-thirds
  • .col-one-quarter
  • .col-three-quarters

Default columns will divide the remaining space automatically.

Responsive

Columns work on mobile by default. If you want columns to be full width (stacked) at tablet onwards then change .row to .row-tablet. For example, the below columns should be full width until resized above 48rem. Resize your browser to see it in ACTION!

.row-tablet / .col-half
.row-tablet / .col-half

Again, if you only want columns on desktop (or wide-desktop), you can use either .row-desktop or .row-wide-destkop.
